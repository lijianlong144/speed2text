# 图像目录

本目录用于存放项目相关的图像文件。

## 需要添加的图像

请添加以下图像文件到此目录：

1. **system_architecture.png** - 系统架构图，用于在项目README中展示
   - 该图应展示AI面试助手的整体架构
   - 包括音频捕获子系统、前端应用、FunASR服务、后端服务和AI服务等组件
   - 参考 `docs/system_architecture.md` 文件中的架构设计

你可以使用任何绘图工具（如Draw.io、Lucidchart、Visio等）创建架构图，然后保存为PNG格式并放置在此目录中。

## 建议的图像尺寸

- 宽度：800-1200像素
- 高度：根据内容调整，保持良好的比例
- 分辨率：72-96 DPI（网页显示）
- 文件格式：PNG（推荐）或JPG
